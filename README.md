# 🎮 Red Light Green Light 3D (OpenGL Game)

A 3D survival game built with **Python + PyOpenGL**, inspired by the
classic *Red Light, Green Light* concept.

You must move forward during **GREEN light** and freeze during **RED
light** --- or the gunmen will shoot you. Reach the finish line before
time runs out to win.

------------------------------------------------------------------------

## 🚀 Features

-   🧠 Intelligent NPCs with dynamic behavior (panic, hesitation, speed
    variation)
-   🚦 Real-time Red Light / Green Light system
-   🔫 Gunmen that shoot moving targets during RED light
-   🪙 Coin system with temporary invisibility power-up
-   👁️ First-person and third-person camera modes
-   🎯 Bullet physics and collision detection
-   🏁 Win/Lose conditions with countdown timer
-   🎮 Smooth player movement and controls

------------------------------------------------------------------------

## 🕹️ Controls

  Key             Action
  --------------- -----------------------------
  W / A / S / D   Move player
  Arrow Keys      Rotate camera / adjust view
  V               Switch camera mode
  C               Toggle cheat mode
  R               Restart game
  Right Click     Toggle camera mode

------------------------------------------------------------------------

## 🧩 Game Rules

-   ✅ Move only during **GREEN light**
-   ❌ Moving during **RED light = instant death**
-   🪙 Collect coins to become invisible for 4 seconds
-   🏁 Reach the finish line within **60 seconds**
-   🎯 Avoid bullets and survive!

------------------------------------------------------------------------

## 🛠️ Technologies Used

-   Python
-   PyOpenGL
-   GLUT (OpenGL Utility Toolkit)

------------------------------------------------------------------------

## 📦 Installation & Setup

1.  Clone the repository: git clone https://github.com/Alif-Al-Fahim/Red_light_Green_light.git
cd Red_light_Green_light

2.  Install dependencies: pip install -r requirements.txt

3.  Run the game: python main.py
